# 🧠 MindCare Indonesia — Platform Edukasi Kesehatan Mental

**MindCare** adalah platform digital yang dirancang untuk menjadi ruang aman, empatik, dan edukatif bagi pemuda Indonesia dalam memahami kesehatan mental. Proyek ini dibangun untuk kompetisi **TECHSOFT 2026 | SOFTCOMPT** dengan fokus pada subtema kesehatan mental.

![MindCare Preview](public/favicon.svg)

## 🌟 Fitur Utama

- **🏠 Home:** Hero section dengan statistik real-time dari WHO & Kemenkes RI.
- **📚 Artikel Edukasi:** Konten berbasis riset tentang Anxiety, Depresi, Stres, dan Self-Care.
- **📝 Self-Check Kuis:** Alat skrining mandiri interaktif untuk mengenali kondisi kesehatan mental (bukan alat diagnosis klinis).
- **🏥 Direktori Layanan:** Daftar rumah sakit jiwa, klinik, dan hotline krisis di berbagai kota di Indonesia.
- **🆘 Tombol Darurat:** Akses cepat ke hotline bantuan profesional (119 ext 8).
- **✨ Animasi Premium:** Pengalaman pengguna yang halus menggunakan `Framer Motion`.

## 🎨 Desain & Estetika

Website ini menggunakan palet warna yang menenangkan dan empatik:
- **Deep Teal (#1A2F3A):** Memberikan kesan mendalam dan stabil.

- **Calm Blue (#A8C5DA):** Memberikan ketenangan.
- **Warm Sand (#F0E6D3):** Memberikan kehangatan pada konten.

**Tipografi:**
- **Heading:** Lora (Serif - Elegan & Hangat)

## 🛠️ Stack Teknologi

- **Core:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Routing:** [React Router v6](https://reactrouter.com/)
- **Icons:** Custom SVG & Emojis

## 🚀 Cara Menjalankan Project Secara Lokal

1. **Clone repository:**
   ```bash
   git clone https://github.com/Sabiladitia20/MindCare.git
   ```

2. **Masuk ke direktori project:**
   ```bash
   cd MindCare
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Jalankan server development:**
   ```bash
   npm run dev
   ```

5. **Buka di browser:**
   Akses `http://localhost:5173/`

## 📊 Kaitan dengan Tema INNOVATE

MindCare mengimplementasikan inovasi digital dalam penyampaian edukasi kesehatan mental. Dengan menggunakan *Single Page Application (SPA)*, transisi antar halaman terasa instan dan interaktif, menghilangkan hambatan akses informasi bagi pengguna yang sedang dalam kondisi emosional sensitif.
